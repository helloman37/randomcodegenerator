FAQ

What is code generator?
It's a free tool for generating unique random codes using letters, digits and other marks.

Who is this generator for?
It's dedicated for people running marketing campaigns, using random promo codes.

How random are the codes?
The longer pattern the . Let’s take 8 characters long strings for example:

charset = 0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ — 63 characters. In this case there are 63⁸ = 248155780267521 possible codes. It means that if you issue a billion codes the probability of guessing a code is 10⁹/63⁸ = 0.000004… — 4 in a million.

What is the maximum amount of codes I can generate?
10 000

What generated codes can be used for?
Generated codes can be used as promo codes, voucher numbers, gift cards numbers, serial numbers, lottery numbers, strong passwords, hard to guess codes etc.

What is code prefix?
First few characters of code. It enables you to assign codes to a particular marketing campaign.

What is code postfix?
Last few characters of code. Useful for reporting.

What is code pattern?
The way the code is made. You can define a pattern using following characters: uppercase, lowercase, digits, special characters, punctuation etc. You can also decide about sequence. For example, you can create pronounceable codes that will be associated with slogans, brand names etc. See example below: FR38_SH1PP2NG_N8W, FR44_SH7PP6NG_N9W

Can I customize the length, prefix and pattern of codes?
Yes, these parameters are customizable.

Which characters can codes contain?
Codes can mix upper cases, lower cases, digits, special characters, punctuation and brackets.

What is the maximum length for generated codes?
Code length is unlimited.

Can I save generated codes as CSV file?
Yes, you copy and paste it to Excel and then save as a CSV file.

Is registration required to create codes?
No, you can use generator without any registration.

Is payment required to create codes?
No, generator is free.

I need more codes that you can generate. What should I do?
You can change one of used parameters (e.g. code length) and run the tool again.

Do you store the generated codes?
No, we don’t store the codes after they have been generated and we don’t have a list of pre-generated codes to choose them from. They are generated at requested time.
